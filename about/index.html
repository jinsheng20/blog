位数从一开始不是从0开始

int bit_mask(int nbits)
{
    int i;
    int ret = 1;
    for(i = 0; i < nbits -1;i++)
    {
        ret = ret * 2+1;
    }
    return ret;
}

Socket CAN测试 出现ip: either “dev” is duplicate, or “type” is garbage 问题
因为新版BusyBox对Socket CAN的不支持，所以为了测试和使用Socket CAN，我们需要自己编译Socket CAN的工具。这里介绍两个工具，分别是iproute2 和 canutils。

iproute2
（1）下载iproute2的新源码http://www.kernel.org/pub/linux/utils/net/iproute2/ 。笔者下载的是iproute2 3.6.0。
（2）解压iproute2-3.6.0.tar.xz，修改Makefile第33行。
33 #CC = gcc
34 CC = arm-none-linux-gnueabi-gcc
（3）因为我们只需要iprout2的ip命令，所以修改Makefile的第42行。
42 #SUBDIRS=lib ip tc bridge misc netem genl man
43 SUBDIRS=lib ip
修改完成执行make命令，生成ip命令(编译会报错这个时候ip命令已经生成位于ip文件夹下)，拷贝到开发板文件系统目录。

设置can 相关命令

关闭can
ifconfig can0 down
设置can 通信参数(设置比特率125000)
ip link set can0 up type can bitrate 125000 triple-sampling on
(ip link set can0 up type can bitrate 125000)

启动can
ifconfig can0 up
显示can 详细信息
/ip -details link show can0

ip命令用来显示或操纵Linux主机的路由、网络设备、策略路由和隧道，是Linux下较新的功能强大的网络配置工具

ip link list 用ip命令显示网络设备的运行状态
